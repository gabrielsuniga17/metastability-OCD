    Este é um repositório criado para a descrição e os resultados do projeto de mestrado intitulado "Dinâmica da conectividade funcional global na resistência as obsessões e compulsões em crianças e adolescentes com TOC", do aluno de pós-graduação em Neurociência e Cognição da Universidade Federal do ABC, Gabriel Suniga, sob orientação do Prof. Dr. Claudinei Biazoli.

    Os arquivos estão separados da seguinte maneira:
- *descritivos_datasets.pdf*: Contém os dados descritivos de ambos os datasets, como medidas descritivas de posição (média, mediana) e dispersão (desvio padrão). São mostrados também os histogramas e boxplots das variáveis Total_YBOCS (pontuação total no YBOCS), Comp_YBOCS (pontuação apenas na categoria compulsão) e Obs_YBOCS (pontuação apenas na categoria obsessão).
- *metaestabilidade.r*: Código em R que calcula a assinatura de metaestabilidade para cada indivíduo de ambos os datasets. Está dividido nas seguintes seções: *bibliotecas* (pacotes utilizados no script); *funções* (principais funções aplicadas nos dados); *importação* (importação dos arquivos outputs do CONN de cada sujeito) e; *resultados* (cálculo da metaestabilidade).
- *pre_processamento.pdf*: Neste arquivo é descrito como foi realizado o pre-processamento, e contém o link para os resultados obtidos. Também introduz o software CONN toolbox, além das etapas de pre-processamento e denoising.
- *resultados_preliminares.pdf*: Apresenta os resultados dos principais testes estatísticos aplicados.
